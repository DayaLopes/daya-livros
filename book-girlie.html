O que você encontrará no Book Girlie?

Organização e Registros

Biblioteca: Espaço visual para colocar as capas dos 120 livros.

Páginas de Resenhas: Espaço dedicado para avaliação individual de cada leitura.

Estante Virtual: Página interativa para "pintar a lombada" e deixar a estante colorida e aesthetic.

TBR (To Be Read): Lista de desejados e fila de leitura.

Coleções

Capa Favorita: Destaque para as capas mais bonitas do ano.

Quotes: Área para guardar as citações preferidas.

Favoritos de Cada Mês: O pódio das melhores leituras mês a mês.

Autores do Ano: Para acompanhar a diversidade de autores lidos.

Calendário e Acompanhamento

Páginas Mensais (Jan a Dez): Com tracker para marcar os dias lidos e espaço para adicionar as capas das leituras do mês.

Recap do Ano: A retrospectiva completa com estatísticas e memórias do ano.

Desafios e Gamificação

Bingo Literário: Uma forma lúdica e criativa de decidir qual será a próxima leitura.

Desafio de A a Z: Para ler livros que completem todas as letras do alfabeto.

Desafio dos Livros: Metas progressivas de quantidade (de 10 até 120 livros).

Leia para Ganhar: Sistema de recompensas para se manter motivada.
